Feel free to experiment but please don’t tweet about Mavo until we launch to the public in November 2016.

You need to include both Mavo’s CSS file and JS file.

  • JS files (pick one)
    • mavo.js: Uncompressed, compatible only with modern browsers
    • mavo.es5.js: Uncompressed, compatible with slightly older browsers
    • mavo.min.js: Compressed, compatible with slightly older browsers
  • CSS file: mavo.css
